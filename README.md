# 📞 Interactive Call Center Performance Dashboard (Salla)

## 🔍 Overview
An advanced business intelligence dashboard engineered in Power BI to analyze call center metrics and operational performance for an e-commerce operation on the Salla platform. It provides actionable insights into agent productivity, customer satisfaction, and call resolution efficiency.

## 🛠️ Key Features & Technical Deliverables
* **Data Transformation & ETL:** Processed and structured raw call logs using Power Query, ensuring optimal data types and handling missing fields.
* **Advanced Data Modeling:** Developed a star schema data model with customized Date and Agent dimensions to support deep analysis.
* **DAX Calculations:** Formulated robust DAX measures to calculate critical KPIs such as Average Handle Time (AHT), Abandonment Rate, and First Call Resolution (FCR).
* **Interactive UI/UX:** Designed a clean, user-friendly dashboard with dynamic slicers for time periods, departments, and agent performance tracking.
